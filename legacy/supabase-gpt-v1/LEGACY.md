# Arquitetura legada Supabase/GPT v1

Este diretório preserva integralmente a arquitetura anterior para referência histórica e técnica. Seu conteúdo não deve ser usado automaticamente em deploy nem por scripts do runtime ativo.

As migrations históricas em `supabase/migrations/` não são autoridade do novo schema. As Edge Functions antigas não fazem parte do novo runtime Node.js, e os contratos OpenAPI preservados não definem automaticamente a API atual.

Conceitos de domínio, nomes estáveis, regras narrativas e exemplos mecânicos podem ser reaproveitados conscientemente após revisão. Nenhum dado ou decisão do legado deve ser tratado como verdade de produção por padrão.
