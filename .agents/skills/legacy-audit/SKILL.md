# Skill: Legacy Audit

## Objetivo

Auditar projeto existente sem alterar arquivos.

## Verificar

- estrutura de pastas;
- package manager;
- scripts;
- stack;
- banco/ORM;
- autenticação;
- testes;
- padrões reais;
- duplicações;
- riscos;
- arquivos sensíveis;
- oportunidades de migração.

## Entrega

- estado atual;
- riscos principais;
- padrões que devem ser preservados;
- duplicações ou problemas;
- plano incremental em etapas pequenas;
- primeira task segura recomendada.

Não reestruture durante a auditoria.
