# Skill: Dependency Evaluation

Antes de instalar dependência, responda:

1. Qual necessidade real?
2. Por que agora?
3. Já existe algo no projeto que resolve?
4. Biblioteca recomendada.
5. Alternativas.
6. Impacto em backend/frontend/banco/testes/deploy.
7. Riscos de manutenção.
8. Comando de instalação.
9. Plano de validação.

Aguarde autorização, salvo se o prompt já autorizou explicitamente.
