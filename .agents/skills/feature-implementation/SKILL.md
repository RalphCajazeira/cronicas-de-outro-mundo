# Skill: Feature Implementation

## Passos

1. Leia `AGENTS.md` e docs/ai relevantes.
2. Entenda o objetivo e o domínio.
3. Procure implementação semelhante.
4. Liste arquivos a tocar.
5. Implemente a menor versão útil.
6. Adicione/ajuste testes quando fizer sentido.
7. Rode validações compatíveis.
8. Entregue resumo com riscos e próximo passo.

## Regras

- Não misturar feature com refactor amplo.
- Não instalar dependência sem autorização.
- Não alterar backend/frontend fora do escopo sem justificar.
